    </>

    Provides Liquid, LiquidPython and LiquidJekyll classes

    Classes
    • Liquid — The main class for external use</>
    • LiquidPython — Support for extended mode of liquidpy</>
    class

    liquid.liquid.Liquid(liquid_template, liquid_config=None, **envs)

    </>

    The main class for external use

    One could provide a config says liquid_config['extended'] = True to switch this object to be initialized as a LiquidPython object.

    Examples 
    >>> liq1 = Liquid("{{a}}")
>>> liq1.__class__ # -> Liquid
>>> liq2 = Liquid("{{a}}", {'mode': 'python'})
>>> liq2.__class__ # -> LiquidPython
>>> isinstance(liq2, Liquid) # -> True
